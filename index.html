About File.info

Detailed information about the linked file types is available in a searchable database of over 10,000 file extensions at File.info. You can discover programmes that open the files and information about unidentified file types online.

File Extensions

A searchable database of more than 10,000 file extensions on File.Info includes comprehensive information about the associated file types. You can discover programmes that open the files and information about unidentified file types online.

The File.info team has been collaborating with both big and small developers to build a central file extensions registry since 2005. Developers can now submit new file extensions and provide details about file types on the reputable website FileInfo.com. Developers can quickly register their file extensions with us.

Software Programs

Additionally, there are evaluations and descriptions of hundreds of software products on File.info. To browse a collection of programmes, go to the Software section. Each software page has a summary of the programme, a feature list, and a list of supported file types.

Using File.info

The search box is located at the top of each page and can be used to check up software products and file extensions. Additionally, you can search the entire website using the Advanced Search function. There is a good possibility that you will find the answer to your file-related questions in the Help Center.
